# Icons

Place your application icons here:

- `32x32.png` - 32x32 pixels
- `128x128.png` - 128x128 pixels
- `128x128@2x.png` - 256x256 pixels for HiDPI
- `icon.icns` - macOS icon
- `icon.ico` - Windows icon
- `icon.png` - Tray icon (recommended: 16x16 or 32x32)

You can generate these icons from a single high-resolution PNG using tools like:
- https://tauri.app/v1/guides/feature/icons/
- ImageMagick: `convert icon.png -resize 32x32 32x32.png`
